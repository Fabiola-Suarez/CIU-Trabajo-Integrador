Objetos

Primero vamos a definir que un objeto es una entidad, algo que existe. Que tiene propiedades que lo definan y lo diferencien, tiene métodos para gestionar sus propiedades y, en general, para manejar a este. Ahora que ya sabemos que es un obeto vamos a mostrarles todos los tipos de objetos que tien javaScript.

Tipos de objetos:

Object Literal: Es una lista compuesta de 0 o más parejas de valores. Cada elemento de la lista este compuesto por una propiedad y un valor. Los elementos de la lista se han de encerrar entre llaves “{ }”. La propiedad se expresa como un literal de tipo texto, y han de cumplir con los nombres validos que se pueden utilizar en javascript. El valor para la propiedad puede ser de cualquier tipo, incluso puede ser otro literal de tipo objeto. Pero mayor desventaja que tiene el Object Literal es que nos permite usar un valor por cada variable, por lo tanto, si necesito un objeto distinto pero con la misma estructura, tengo que crear otro objeto. Por eso este tipo de objeto es la manera "antigua" de hacer las cosas porque javaScript no tenía soporte a objetos, entonces se hacía de esa manera y se metía en algo llamado Prototypes que vamos a ver mas adelante.

Object Constructor: Si necesitamos muchos objetos del mismo tipo tenemos que usar este tipo de objeto.

Object Destructuring: Este objeto nos da una sintaxis para acceder a los valores de un objeto de una forma mas acotada. Este se encarga de buscar el valor si uso el mismo nombre del atributo entre llaves al definir la variable.

Object Literal Enhacement: Es lo contrario al Object Destructuring , este nos permite unir variables a un objeto.
