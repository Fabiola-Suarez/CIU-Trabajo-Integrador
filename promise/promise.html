Promise

Esta herramienta es usada para codigo asíncronas. Una promesa representa un valor que puede estar disponible ahora, en el futuro, o nunca. Cuando hacés un pedido asincrónico a una API tenés dos posibilidades:
1 - Que salga todo bien "resolve".
2 - Que algo salga mal "reject".
Para que vean como son los resultados usamos una API que es pública de randomuser.me/api que te baja un JSON con usuarios random.
