Operaciones con Arrays

Primero vamos a definir que es un array para luego mostrar algunas de sus operaciones.
Un array es una cadena de objetos similares a una lista cuyo proposito es proporcionar métodos para efectuar operaciones de recorrido y mutación. La diferencia que tienen con las listas es que los array son de una longitud fija de elementos, que se determina cuando se inicializa este.

Operaciones:

Spread Operator: Es una operacion que implemeto javaScript, que son tres puntitos (...) que nos da diferentes funcionalidades como duplicar arrays o hacer operaciones para trabajar con estos.

Metodo Reverse: Este metodo da vuelta el array y nos devuelve el ultimo dato (antes de la inversión)

Filter: Esta operacion filtra los elementos generando un nuevo array con todos los elementos que cumplen una determinada condición.

Find: Esta operacion nos devuelve todos los datos de un elemento en especial.

Reduce:Es una operacion magica, porque con pocas lineas de codigo y una funcion dentro de un contenedor, podemos hacer grandes operaciones.

Ahora te mostraremos en codigo como se usan estas operaciones.
